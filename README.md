LoopStuff App
A modern mobile shopping app built using React Native (Expo) and Expo Router.
The app allows users to browse products, add items to a cart, and manage their account with a clean, minimal UI.

✨ Features
🏠 Browse products from API (Fake Store API)
🛒 Add to Cart functionality
💾 Persistent cart using AsyncStorage
👤 User Authentication (Supabase)
🎁 Referral page UI
📦 Listings page
📱 Bottom tab navigation with icons
🎨 Clean and simple UI design

🛠️ Tech Stack
React Native (Expo)
Expo Router
AsyncStorage
Supabase (Authentication)
Fake Store API
Expo Vector Icons (Ionicons)
